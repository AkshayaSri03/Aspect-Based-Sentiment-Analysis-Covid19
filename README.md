# Aspect-Based Sentiment Analysis of COVID-19 Twitter Discussions

## Project Overview
This project performs Aspect-Based Sentiment Analysis (ABSA) on COVID-19 Twitter data using transformer-based NLP models.

## Technologies Used
- Python
- Hugging Face Transformers
- Google Colab
- NLP & Deep Learning

## Repository Contents
- `notebooks/` – Jupyter notebooks for ABSA pipeline
- `datasets/` – Processed datasets used for analysis

## Academic Note
This repository contains the Python/NLP component of a final-year academic project.
